## Preparando o Ambiente
* Visual Studio Code

> yarn
* Vai baixar todas a dependências do projeto.
* Depois é só dar F5 no Visual Studio Code.

* tsc:clean   
`Remove a pasta dist gerada pelo Typescript`

* tsc  
`Efetua o build do typescript`
